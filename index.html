Who's using this device

Modules

Turn off the sections you don't use — they disappear from the menu. Nothing is deleted; switch a module back on and your data is exactly where you left it.

How the app is being used

A read-only look at what each profile is actually logging — useful for working out which parts of Daylign are worth improving. This never loads anyone else's data into the app, so it cannot change their numbers.

Backup your data

Your data syncs to the cloud automatically, but the cloud keeps no history. Download a backup regularly so you always have a copy you control. Save it somewhere safe (Files, iCloud, email).

Restore from a backup

This replaces everything — on this device and in the cloud (all your devices) — with the contents of a backup file. Only use it to recover lost data. Restoring an older file will delete anything newer. A safety copy of your current data is downloaded first, and you'll be asked to type RESTORE to confirm.
